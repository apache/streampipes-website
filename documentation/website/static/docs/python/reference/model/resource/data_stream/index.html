Data stream

DataStream ¶

Bases: Resource

Implementation of a resource for data streams. This resource defines the data model used by resource container (model.container.DataStreams). It inherits from Pydantic's BaseModel to get all its superpowers, which are used to parse, validate the API response and to easily switch between the Python representation (both serialized and deserialized) and Java representation (serialized only).

convert_to_pandas_representation() ¶

Returns the dictionary representation of a data stream to be used when creating a pandas Dataframe.
